Variáveis

  • Variáveis são responsáveis por guardar dados na memória
  • Você pode utilizar var, let ou const
  • Variáveis evitam repetições
  1. Os nomes podem inicial com $, _ ou letras
  2. Procure utilizar CamelCase
  3. Case sensitive: nome é diferente de Nome
  4. Não utilize palavras reservadas

Hoisting

A variável é identificada ANTES da compilação, mas o seu valor só é atribuido durante o processo de compilação.

Mudar o valor atribuído

Só é possível mudar o valor da variável com var e let
