1. O que é uma variável?

Uma forma de armazenar dados sendo utilizado para guardar números e palavras e que podem ser alteradas em determinado momento.

2. Como declarar uma variável numérica inteira ou real usando JS?

Inteira - var primeiroNumero = parseInt(10)

Real - var segundoNumero = parseFloat(10.543)

alert (primeiroNumero) - alert (segundoNumero)

3. Quais são os tipos primitivos do JS?

number, string, boolean, null e function(){}

4. O que significa colocar o valor null dentro de uma variável?

Indica que a variável tem valor ausente, nulo ou vazio
